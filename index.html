By Chenhw

对于其字符串进行操作

1、连接函数paste。

输入命令行

a <- "Hello"

b <- 'How'

c <- "are you? "

print(paste(a,b,c))

print(paste(a,b,c, sep = "-"))

print(paste(a,b,c, sep = "", collapse = ""))

输出结果：

[1] "Hello How are you? "

[1] "Hello-How-are you? "

[1] "HelloHoware you? "

2、格式化数字和字符串 - format()函数

format(x, digits, nsmall, scientific, width, justify = c(“left”, “right”, “centre”, “none”))

x是向量输入。

digits是显示的总位数。

nsmall是小数点右边的最小位数。

科学设置为TRUE以显示科学记数法。

width指示通过在开始处填充空白来显示的最小宽度。

justify是字符串向左，右或中心的显示。

输入命令行

result <- format(23.123456789, digits = 9)

print(result)

result <- format(c(6, 13.14521), scientific = TRUE)

print(result)

result <- format(23.47, nsmall = 5)

print(result)

result <- format(6)

print(result)

result <- format(13.7, width = 6)

print(result)

result <- format("Hello", width = 8, justify = "l")

print(result)

result <- format("Hello", width = 8, justify = "c")

print(result)

输出结果

[1] "23.1234568"

[1] "6.000000e+00" "1.314521e+01"

[1] "23.47000"

[1] "6"

[1] " 13.7"

[1] "Hello "

[1] " Hello "

3、计算字符串中字符的个数

x <-c(“R”, “Rstudio”, “Revolution R”)

返回3，向量x中共有3个元素

length(x)

返回1 7 12，向量x中每个元素的字符个数

nchar(x)

x代表向量

输入

result <- nchar("Count the number of characters")

print(result)

输出

[1] 30

4、将字符串全部变成大写和小写

toupper(x)

tolower(x)

输入

result <- toupper("Changing To Upper")

print(result)

result <- tolower("Changing To Lower")

print(result)

输出

[1] "CHANGING TO UPPER"

[1] "changing to lower"

5、提取字符串的一部分 - substring()函数

substring(x,first,last)

x是字符向量输入。

首先是要提取的第一个字符的位置。

last是要提取的最后一个字符的位置。

输入

result <- substring("Extract", 5, 7)

print(result)

结果

[1] "act"

条形图（括号中的内容为属性，只要在正常调用的时候加上属性等于什么什么就可以了）

barplot(H, xlab, ylab, main, names.arg, col)

H是包含在条形图中使用的数值的向量或矩阵。

xlab是x轴的标签。

ylab是y轴的标签。

main是条形图的标题。

names.arg是在每个条下出现的名称的向量。

col用于向图中的条形提供颜色。

H是必须的，其余可以不写

箱线图

boxplot(x, data, notch, varwidth, names, main)

x是向量或公式。

数据是数据帧。

notch是逻辑值。 设置为TRUE以绘制凹口。

varwidth是一个逻辑值。 设置为true以绘制与样本大小成比例的框的宽度。

names是将打印在每个箱线图下的组标签。

main用于给图表标题。

直方图

hist(v,main,xlab,xlim,ylim,breaks,col,border)

v是包含直方图中使用的数值的向量。

main表示图表的标题。

col用于设置条的颜色。

border用于设置每个条的边框颜色。

xlab用于给出x轴的描述。

xlim用于指定x轴上的值的范围。

ylim用于指定y轴上的值的范围。

break用于提及每个条的宽度

折线图

plot(v,type,col,xlab,ylab)

v是包含数值的向量。

类型采用值“p”仅绘制点，“l”仅绘制线和“o”绘制点和线。

xlab是x轴的标签。

ylab是y轴的标签。

main是图表的标题。

col用于给点和线的颜色

/*通过使用lines(里面加向量)函数，可以在同一个图表上绘制多条线*/

散点图

plot(x, y, main, xlab, ylab, xlim, ylim, axes)

x是其值为水平坐标的数据集。

y是其值是垂直坐标的数据集。

main要是图形的图块。

xlab是水平轴上的标签。

ylab是垂直轴上的标签。

xlim是用于绘图的x的值的极限。

ylim是用于绘图的y的值的极限。

axes指示是否应在绘图上绘制两个轴

饼状图

pie(x, labels, radius, main, col, clockwise)

x是包含饼图中使用的数值的向量。

labels用于给出切片的描述。数组，分别对应每一个块的名称

radius表示饼图圆的半径（值-1和+1之间）。

main表示图表的标题。

col表示调色板。

clockwise是指示片段是顺时针还是逆时针绘制的逻辑值。

JavaScript对于dom节点的基本操作

By Chenhw

Published 2017-08-28

一：始终记住DOM是一个树形结构。操作一个DOM节点实际上就是这么几个操作：

更新：更新该DOM节点的内容，相当于更新了该DOM节点表示的HTML的内容；

遍历：遍历该DOM节点下的子节点，以便进行进一步操作；

添加：在该DOM节点下新增一个子节点，相当于动态增加了一个HTML节点；

删除：将该节点从HTML中删除，相当于删掉了该DOM节点的内容以及它包含的所有子节点。
在操作一个DOM节点前，我们需要通过各种方式先拿到这个DOM节点

最常用的方法是document.getElementById()【返回一个节点】和document.getElementsByTagName()【返回一组节点】， 以及CSS选择器document.getElementsByClassName()【返回一组节点】

可以先对于节点进行取，然后再通过class或tagname去取其中的节点组

二：更新DOM 通过修改innerHTML属性，这个方式非常强大，不但可以修改一个DOM节点的文本内容 ，还可以直接通过HTML片段修改DOM节点内部的子树。内部要改字符串的可以使用单引号括起来。 可以使用

var p = document.getElementById('p-id'); p.style.color = '#ff0000';来改变当前节点的CSS值。

ps：如果有连接符则把连接符后的第一个字母大写

三：插入DOM 静态插入节点：

var js = document.getElementById('js');

list = document.getElementById('list');

list.appendChild(js); 此操作会把id为js的位置换到list这个节点后的最后一个 动态插入节点：

var list = document.getElementById('list'); new= document.createElement('p');/*创造一个新的节点*/

new.id = 'hhhhh';/*给这个节点赋值*/

new.innerText = 'hhhh';/*给这个节点它的内部text*/

list.appendChild(new);/*同样的操作把此id的节点放入list的最后*/

同样的可以动态加入新的CSS类：

var d = document.createElement('style');

d.setAttribute('type', 'text/css');

d.innerHTML = 'p { color: red }';

document.getElementsByTagName('head')[0].appendChild(d);/*会把这个节点放入head的最后的地方*/

如果想要在指定的地方进行操作则有：

ref = document.getElementById('front')； list.insertBefore(new, ref);/*此操作会把new节点放入ref的前面*/

四：删除DOM 要删除一个节点，首先要获得该节点本身以及它的父节点，然后，调用父节点的removeChild把自己删掉 // 拿到待删除节点:

var self = document.getElementById('to-be-removed'); // 拿到父节点:

var parent = self.parentElement; // 删除:

var removed = parent.removeChild(self);

removed === self; // true 个人建议删除节点从后向前删除，因为删除之后数组会向前进位，所以会改变其位置
