WEB Storage的一些简单的应用方法

var storage = window.localStorage;
storage.setItem("b","123456");//在这里是对于b这个storage中的变量进行赋值操作
storage.getItem("b");//这里是对于storage中的b的值进行提取
storage.removeItem("b");//这里就是对于b进行的一个清空

整体的storage我们可以看出来是和cookie有很多的类似之处的，但是这里没有expire这个量，即认为我们可以永久存放这些变量，至少是长时间的，而且在上一篇也说到了cookie的存储量大概至是在4KB左右，而目前HTML5所支持的local的storage的大小达到了5MB，这个空间存放一些比较基础的东西是已经足够了的。
如果是想清除所有的键值对，则需要使用storag.clear()函数。

function showStorage(){
 for(var i=0;i<storage.length;i++){
  //key(i)获得相应的键，再用getItem()方法获得对应的值
  document.write(storage.key(i)+ " : " + storage.getItem(storage.key(i)) + "<br>");
 }
}

key(i)是为了获取storage中的变量名，这样就可以看到所有本地储存的内容。
