表的名称和字段名 需要我们自己定义名称

SQL不区分大小写（命名）但是表名就是文件名，windows不区分大小写，linux区分

1、一定要有意义（英文或英文的组合和多个单词的缩写）

2、自己定的名称最好都小写

3、sql语句都大写

二、数据值和列类型

1、数值型

整型（整数）/*细分都是按照空间大小来分的可以放的下就可以了*/

非常小的整型 1字节

较小的整型 2字节

中等大小的整型 3字节

标准整型 4字节

大整数型 8字节

浮点型（小数）

float(H,D) 4字节(H和D都可被指定

double(H,D) 8字节

定点数

decimal（H,D） H+2字节

2、字符型

小的：char 255（固定长度字符串）即以4个字节为单位存，处理速度比varchar快的多

varchar 255（可变长度字符串）即可以变，单永远是内容加1个字节，像文章标题之类不确定长度的使用varchar（更多的被使用）

大的：text处理文本 2^16-1

blob处理相片 2^16-1

ENUH 枚举 1或2个字节 一次只能存一个值

set集合 1，2，4，8个字节一次可以存多个值，用“，”隔开

3、日期型

DATE YYYY-MM-DD

TIME hh-mm-ss

DATETIME YYYY-HH-DD hh-mm-ss

TIMESTAMP YYYYHHDDhhmmss

YEAR YYYY

4、NULL

三、具体操作

1、创建库：create database +名字

2、展示库：show databases

3、创建表：create table +名字

4、查看表的。。。DESC + 表名

5、创建表的列名：create table +表名（

id int（10），

name char（100），

qq int（10））；

6、信息的插入：insert into 表名（id，name，qq。。。）value（1，大明，1206095865）；

7、信息的检索： select * from 表名；（此处返回所有表的内容，但是对于大部分的情况来说需要具体的查找）

select * from 表名 where name="小明"；（返回小明所有的信息）

select name from 表名；（返回那一列）

select *from 表名 where name like "大%"；（模糊匹配，后面的百分号代表任意值）

同理 select name, id from 表名 where name like "%大%"；（即为返回带“大”字的name和id)

select 中同时可以包含语句，比如if之类的，但是如果不改名，则把整个的语句当作列名，如在语句后加一个as stuhhhh,则stuhhhh就是它那个列的名字（别名）

select *from 表名 where name like "大%" and id="1"；（其中加入了逻辑运算符and，同理“or”）

select concat("姓名"+name+"序号"+id+"QQ"+qq) as stuinformation from stu;(把字符串串联起来)
