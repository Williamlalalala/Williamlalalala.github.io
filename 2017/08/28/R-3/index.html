对于其字符串进行操作

1、连接函数paste。

输入命令行

a <- "Hello"

b <- 'How'

c <- "are you? "

print(paste(a,b,c))

print(paste(a,b,c, sep = "-"))

print(paste(a,b,c, sep = "", collapse = ""))

输出结果：

[1] "Hello How are you? "

[1] "Hello-How-are you? "

[1] "HelloHoware you? "

2、格式化数字和字符串 - format()函数

format(x, digits, nsmall, scientific, width, justify = c(“left”, “right”, “centre”, “none”))

x是向量输入。

digits是显示的总位数。

nsmall是小数点右边的最小位数。

科学设置为TRUE以显示科学记数法。

width指示通过在开始处填充空白来显示的最小宽度。

justify是字符串向左，右或中心的显示。

输入命令行

result <- format(23.123456789, digits = 9)

print(result)

result <- format(c(6, 13.14521), scientific = TRUE)

print(result)

result <- format(23.47, nsmall = 5)

print(result)

result <- format(6)

print(result)

result <- format(13.7, width = 6)

print(result)

result <- format("Hello", width = 8, justify = "l")

print(result)

result <- format("Hello", width = 8, justify = "c")

print(result)

输出结果

[1] "23.1234568"

[1] "6.000000e+00" "1.314521e+01"

[1] "23.47000"

[1] "6"

[1] " 13.7"

[1] "Hello "

[1] " Hello "

3、计算字符串中字符的个数

x <-c(“R”, “Rstudio”, “Revolution R”)

返回3，向量x中共有3个元素

length(x)

返回1 7 12，向量x中每个元素的字符个数

nchar(x)

x代表向量

输入

result <- nchar("Count the number of characters")

print(result)

输出

[1] 30

4、将字符串全部变成大写和小写

toupper(x)

tolower(x)

输入

result <- toupper("Changing To Upper")

print(result)

result <- tolower("Changing To Lower")

print(result)

输出

[1] "CHANGING TO UPPER"

[1] "changing to lower"

5、提取字符串的一部分 - substring()函数

substring(x,first,last)

x是字符向量输入。

首先是要提取的第一个字符的位置。

last是要提取的最后一个字符的位置。

输入

result <- substring("Extract", 5, 7)

print(result)

结果

[1] "act"

下面看对于向量的操作

向量的操作

向量是最基本的R语言数据对象，有六种类型的原子向量。 它们是逻辑，整数，双精度，复杂，字符和原始。

1、 对数值数据使用冒号运算符

输入

Creating a sequence from 5 to 13.

v <- 5:13

print(v)

Creating a sequence from 6.6 to 12.6.

v <- 6.6:12.6

print(v)

If the final element specified does not belong to the sequence then it is discarded.

v <- 3.8:11.4

print(v)

输出

[1] 5 6 7 8 9 10 11 12 13

[1] 6.6 7.6 8.6 9.6 10.6 11.6 12.6

[1] 3.8 4.8 5.8 6.8 7.8 8.8 9.8 10.8

2、sequence (Seq.)序列运算符

输入

print(seq(5, 9, by = 0.4))

输出

[1] 5.0 5.4 5.8 6.2 6.6 7.0 7.4 7.8 8.2 8.6 9.0
