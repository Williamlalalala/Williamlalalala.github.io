JS的增加节点的操作

我们在进行前后端交互的时候经常需要实现一些功能，而为了不使整个页面需要再次刷新，我们需要使用js来增加节点，并在触发事件的时候进行增加相应的东西，实现异步刷新

我先做了一个非常简单的框加几个文本，这样一会会看的很清楚

<input type="button" id="chenhw" name="">
<p id="in">hhhhh</p>
<p id="">hhhhh</p>
<p id="">hhhhh</p>

然后js代码如下

window.onload=function(){//在这里我觉得加不加这个都差不多
    var one=document.getElementById('chenhw');//获取节点
    one.onclick=function(){//节点被click的时候触发
        insert();//调用这个函数
    }
}
function insert(){
    var start=document.getElementById('in');//获取想操作的节点位置
    var add=document.createElement('create');//create一个节点
    add.innerHTML="lalal";//改变这个节点的内容
    start.insertBefore(add, null);//插入
}

这样就可以看到我们在click之后整个页面会做出相应的操作
